## Style Modifiers

Add these to any prompt for consistency:

**For darker/moodier:**
```
, void black background, minimal lighting, deep shadows, noir atmosphere, isolated figure
```

**For more energy:**
```
, dynamic pose, motion blur, particle effects, neon bloom, high contrast, aggressive angle
```

**For album art style:**
```
, square composition, bold graphic design, text integration, high contrast, print-ready, iconic imagery
```

**For live performance:**
```
, concert lighting, fog machine haze, crowd silhouettes, stage setup, LED panels, spectacular scale
```
